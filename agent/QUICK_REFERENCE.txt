└────────┬────────┘
         │
         ▼ (feedback loop)
     [Retrain]

╔═══════════════════════════════════════════════════════════════════╗
║ 📊 THE RL CYCLE                                                   ║
╚═══════════════════════════════════════════════════════════════════╝

1️⃣  OBSERVE   → Gather 48 data points from providers
2️⃣  PREDICT   → Neural network outputs probability distribution
3️⃣  ACT       → Execute trade if confidence > threshold
4️⃣  REWARD    → Calculate P/L and update neural weights

╔═══════════════════════════════════════════════════════════════════╗
║ 🔧 KEY PARAMETERS                                                 ║
╚═══════════════════════════════════════════════════════════════════╝

In brain.py:
  learning_rate        = 0.001   (How fast agent learns)
  gamma               = 0.95    (Future reward discount)
  epsilon             = 0.10    (Exploration rate 10%)
  max_memory_size     = 1000    (Experience replay buffer)

In predictive_agent.py:
  confidence_threshold = 0.70    (Min confidence to trade)
  trade_amount_usdc    = 10.0    (Trade size)
  simulation_mode      = True    (Safe testing mode)
